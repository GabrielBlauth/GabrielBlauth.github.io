Task 1

Create three variables with your first name, last name, and favorite color. Use a template literal to output: "Hi! My name is [firstName] [lastName] and my favorite color is [color]."

Task 2

Create a variable with the text: "JavaScript is fun!"
- Output its total length
- Output the first character using [] notation
- Output the last character using [] notation
- Output the start position of the word "fun" using [] notation

Task 3

Start with the text: "The quick brown fox"
- Extract just the words "quick brown"
- Replace "fox" with "dog"
- Add the text "jumps over the lazy dog" to the end

Task 4

Create a temperature converter that takes a Celsius temperature and converts it to Fahrenheit using this formula: (C × 9/5) + 32.

Task 5

Create a function with parameters that calculates the area of a rectangle. The function should return its output to the calling program and display the result in the web browser console.

Task 6

Create a script with two string variables, username and password. Use the AND (&&) operator to check if:
- The username is "student123" AND
- The password is "webdev2024"
In the console, display "Access granted" if both conditions are met, otherwise display "Access denied"

Task 7

Write a script that checks a temperature number value and displays different messages in the console:
- If temperature is above 30, display "Heat warning!"
- If temperature is between 25-30, display "Warm weather"
- If temperature is between 15-24, display "Pleasant weather"
- If temperature is below 15, display "Cool weather"

Task 8

Write a function that takes any value and tells you if it's falsy:
function isFalsy(value) {
// Your code here
}
The function should return true if the value is falsy and false if it's truthy.
Test your function with: 0, "", null, undefined, NaN, false

Task 9

Add a <div> with a unique id and some sample text.
Add two buttons labeled "Turn Red" and "Reset Color."
When "Turn Red" is clicked, use an event handler function to change the background color of the <div> to red.
When "Reset Color" is clicked, restore the original background color.

Task 10

Create a button labeled "Add Paragraph."
When the button is clicked, use JavaScript event handler to create a new <p> element with the text "This is a new paragraph!" and append it to the page.

Task 11

Create a button labeled "Add Bold Text."
Create a <p> element with the initial text "This is some text."
When the button is clicked, update the paragraph to include the text "This is some bold text."

Task 12

Create an object called objAlbum with the following properties and output each property to the console:
- title (string)
- artist (string)
- year (number)
- isAvailable (boolean)

Task 13

Create an array called arrColours with five different colour names. Then, in the console, output:
- The first colour
- The last colour using the array's length
- The total number of colours in the array

Task 14

Create an array called arrAlbums with at least four album objects. Each album object should have these properties:
- title (string)
- artist (string)
- year (number)
- isRead (boolean)
You may use AI to generate the details.
